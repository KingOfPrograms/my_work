## 二、投注策略设计

### 2.1 策略框架："共识 + 凯利 + 让球"

```
筛选层 (共识门槛)
  ↓
估值层 (期望值计算)
  ↓
仓位层 (凯利公式)
  ↓
执行层 (让球盘优化赔率)
```

### 2.2 第一层 — 共识筛选

**仅投注 5/6 或 6/6 共识的比赛。**

- 6/6 全票: 历史准确率 66.7%，真实概率估算 60-65%
- 5/6 共识: 历史准确率 60.0%，真实概率估算 55-58%

不符合共识门槛的比赛 **弃投**。

### 2.3 第二层 — 期望值计算

```
EV = P_win × Odds - 1

其中:
  P_win = 模型共识估算的真实概率 (保守折后)
  Odds  = 彩票赔率（欧洲赔率）
```

**投注条件: EV > 0.05**（即期望回报 > 5%）

保守概率估算（考虑模型集体偏见）：
- 6/6 共识 → P = 60%（历史 66.7% 打八折，留安全边际）
- 5/6 共识 → P = 52%（历史 60% 打八五折）

### 2.4 第三层 — 凯利公式仓位管理

```
f* = (P × Odds - 1) / (Odds - 1)

f* = 理论最优投注比例（占总资金）
实际投注 = f* × 0.25  (四分之一凯利，降低波动)
单注上限 = 5% 总资金 (硬性风控)
```

### 2.5 第四层 — 让球盘优化

当 6/6 共识预测某队必胜时，直接买胜赔率通常极低（~1.2-1.5），不值得投。

**使用让球盘提升赔率：**
- 模型预测强队净胜 1 球以上 → 买让球胜（-1），赔率升至 ~2.0-3.0
- 历史数据显示此类比赛大胜概率高（如德国 7-1）
